## Dependencies

- OpenCV4.5
- Percipio SDK
- PaddleInference
- QT5
- C++ Standard 17



## Procedures
1. Initialization
2. Load the image and perform image segmentation
3. The packages are detected with a segmented mask
4. Always pick the first one and track it
